# Requisitos do Projeto

Este documento descreve as funcionalidades básicas do sistema que será desenvolvido na pasta `src/main.py`.

## 功能 Requisitos Funcionais (O que o sistema deve fazer)
1. O sistema deve permitir a entrada de dados do usuário.
2. O sistema deve processar as informações de acordo com a lógica definida no escopo da atividade.
3. O sistema deve exibir um resultado claro no terminal.
